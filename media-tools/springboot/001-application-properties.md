# application-properties


## application.properties

```properties
server.port=3000
spring.application.name=media-api
spring.datasource.url=jdbc:postgresql://localhost:5432/backend_media
```

---

## application.yml


```yaml
server:
  port: 3000

spring:
  application:
    name: media-api
  datasource:
    url: jdbc:postgresql://localhost:5432/backend_media
    username: postgres
    password: postgres
```
