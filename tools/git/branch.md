# Git – Branches

## Création / Renommage

| Action               | Commande                          |
|----------------------|-----------------------------------|
| Créer une branche    | `git checkout -b feature-01`      |
| Renommer une branche | `git branch -M feature-02`        |
| Supprimer une branche      | `git branch -d feature-02`|
| Changer de branche          | `git checkout feature-01` |

## Utilisation / Gestion

| Action                      | Commande                   |
|-----------------------------|----------------------------|
| Lister branches locales    | `git branch`              |
| Lister branches distantes  | `git branch -r`           |
| Lister toutes les branches | `git branch -a`           |
