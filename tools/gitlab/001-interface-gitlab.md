# Interface gitlab

## Changer la langue en **English** (compte utilisateur)

---

## Étapes

1. Cliquer sur ton **avatar** (en haut à droite).
2. Ouvrir **Preferences**.
3. Aller dans la section **Localization** (ou **Language** selon la version).
4. Sélectionner **English** dans la liste.
5. Cliquer sur **Save changes**.
