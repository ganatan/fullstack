# Désactiver la minimap dans VS Code

La zone à droite dans l’éditeur est la **minimap**.
Pour la masquer :
**View** → **Appearance** → décocher **Minimap**.

Autre méthode :
`Ctrl + ,` puis rechercher `minimap`
et désactiver **Editor: Minimap Enabled**.

Dans `settings.json` :
`"editor.minimap.enabled": false`
