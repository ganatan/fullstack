# Git – Branches

| Action                     | Commande                          |
|----------------------------|-----------------------------------|
| Renommer une branche       | `git branch -M main`              |
| Changer de branche         | `git checkout feature-001`        |
| Créer une branche          | `git checkout -b feature-001`     |
| Supprimer une branche      | `git branch -d feature-001`       |
| Lister branches locales   | `git branch`                      |
| Lister branches distantes | `git branch -r`                   |
| Lister toutes les branches| `git branch -a`                   |
