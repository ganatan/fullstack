# Spring

- **Java** : langage de programmation.
- **Java SE 8** : version du langage encore très utilisée.
- **Servlet** : base du web Java (requête → réponse).
- **Tomcat** : serveur web qui exécute les Servlets.
- **Spring** : framework qui crée et gère les objets.
- **ApplicationContext** : conteneur des objets Spring.
- **Spring MVC** : module Spring pour le web.
- **DispatcherServlet** : servlet centrale qui route les requêtes.

## Flux

```
Navigateur → Tomcat → DispatcherServlet → Controller → Réponse
```

- Spring MVC repose sur les Servlets mais en simplifie l’usage.
