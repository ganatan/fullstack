# Java Starter

## Installation Java

```bash
java -version
javac -version
```

## Création projet IntelliJ

```
Name: java-starter
Build system: IntelliJ
JDK: 21
```

## Structure projet

```
java-starter
 └─ src
    └─ Main.java
```

## Premier programme Java

```java
public class Main {

  public static void main(String[] args) {
    System.out.println("Bonjour Java");
  }

}
```

## Exécution

```
Bonjour Java
```

## Commandes compilation manuelle

```bash
javac Main.java
java Main
```
