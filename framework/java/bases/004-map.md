# Map

Tous les exemples sont exécutables directement dans un contrôleur Spring Boot.

---

## Code

```java
package com.ganatan.starter.api.root;

import java.util.HashMap;
import java.util.Map;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class RootController {

  @GetMapping("/map")
  public Map<String, Object> root() {

    Map<String, Integer> hashMap = new HashMap<>();
    hashMap.put("one", 1);
    hashMap.put("two", 2);
    hashMap.put("three", 3);

    Integer value = hashMap.get("two");

    boolean containsKey = hashMap.containsKey("three");

    hashMap.remove("one");

    int size = hashMap.size();

    Map<String, String> immutableMap = Map.of(
      "A", "alpha",
      "B", "beta"
    );

    return Map.of(
      "hashmap", hashMap,
      "get_value", value,
      "contains_key", containsKey,
      "size", size,
      "immutable_map", immutableMap
    );
  }
}
```

---

## Principes clés

- `Map` stocke des couples clé / valeur
- `HashMap` n’est pas ordonnée
- `get` retourne `null` si la clé n’existe pas
- `containsKey` évite les `NullPointerException`
- `Map.of` crée une map immuable
