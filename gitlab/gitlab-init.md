# gitlab-init.md

## Create repository

Menu:
- En haut à droite de GitLab → bouton **+**
- **New project**
- **Create blank project**

Paramètres :
- Project name: test-EXAMPLE01
- Visibility: Private

---

## Add maintainer

Menu:
- Ouvrir le projet `test-EXAMPLE01`
- Menu gauche → **Settings**
- **General**
- Descendre jusqu’à **Project members**
- **Invite members**

Paramètres :
- Username: <gitlab-username>
- Role: Maintainer

---

## Remove member

Menu:
- Projet `test-EXAMPLE01`
- **Settings**
- **General**
- **Project members**

Action :
- Menu **⋯** sur le membre
- **Remove member**

---

## Emails language

Menu:
- Avatar (en haut à droite)
- **Preferences**
- **Localization**

Paramètre :
- Language: English
