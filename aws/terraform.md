# Glossaire
  Terraform           outil Infrastructure as Code déclaratif
  HashiCorp           éditeur de Terraform
  HCL                 langage déclaratif utilisé par Terraform
  .tf                 fichier de configuration Terraform
  Provider            plugin qui parle à une API (AWS, Kubernetes, Helm…)
  Resource            objet créé et géré par Terraform
  Data source         objet existant, lecture seule
  State (tfstate)     vérité de l’infrastructure déployée
  Plan                simulation des changements
  Apply               application réelle des changements
