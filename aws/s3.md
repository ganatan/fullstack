**Amazon S3 (Simple Storage Service)** est un service de stockage d’objets d’AWS.  

  Il stocke des fichiers sous forme d’objets dans des **buckets** globalement uniques.  
  Il est hautement durable, chiffré et accessible via API HTTP.  
  S3 est utilisé pour les backups, artefacts CI/CD, fichiers statiques et **state Terraform**.  
  Il ne fournit ni système de fichiers, ni base de données, ni requêtes.
