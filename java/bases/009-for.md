# for – Java (essentiel)

---

## Code

```java
package com.ganatan.starter.api.root;

import java.util.ArrayList;
import java.util.List;
import java.util.Map;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class RootController {

  @GetMapping("/for")
  public Map<String, Object> root() {

    List<Integer> classicFor = new ArrayList<>();
    for (int i = 0; i < 5; i++) {
      classicFor.add(i);
    }

    List<Integer> decrementFor = new ArrayList<>();
    for (int i = 5; i > 0; i--) {
      decrementFor.add(i);
    }

    List<Integer> multipleVars = new ArrayList<>();
    for (int i = 0, j = 10; i < 3; i++, j -= 2) {
      multipleVars.add(i * 100 + j);
    }

    List<Integer> enhancedFor = new ArrayList<>();
    int[] values = {1, 2, 3, 4, 5};
    for (int v : values) {
      enhancedFor.add(v);
    }

    List<Integer> breakCase = new ArrayList<>();
    for (int i = 0; i < 10; i++) {
      if (i == 4) {
        break;
      }
      breakCase.add(i);
    }

    List<Integer> continueCase = new ArrayList<>();
    for (int i = 0; i < 6; i++) {
      if (i % 2 == 0) {
        continue;
      }
      continueCase.add(i);
    }

    List<Integer> nestedFor = new ArrayList<>();
    for (int outer = 1; outer <= 3; outer++) {
      for (int inner = 1; inner <= 2; inner++) {
        nestedFor.add(outer * 10 + inner);
      }
    }

    int sum = 0;
    for (int v : values) {
      sum += v;
    }

    return Map.of(
      "classic_for", classicFor,
      "decrement_for", decrementFor,
      "multiple_vars", multipleVars,
      "enhanced_for", enhancedFor,
      "break", breakCase,
      "continue", continueCase,
      "nested_for", nestedFor,
      "sum_enhanced_for", sum
    );
  }
}
```

---

## Résumé

- `for(init; condition; update)` : boucle contrôlée par compteur
- `for` peut décrémenter et gérer plusieurs variables
- `for-each` itère sur un tableau/collection
- `break` sort de la boucle
- `continue` saute l’itération courante
- boucles imbriquées possibles
