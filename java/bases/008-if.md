# if

---

## Code

```java
package com.ganatan.starter.api.root;

import java.util.Map;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class RootController {

  @GetMapping("/if")
  public Map<String, Object> root() {

    int value = 5;

    String simpleIf;
    if (value > 3) {
      simpleIf = "greater than 3";
    } else {
      simpleIf = "less or equal to 3";
    }

    String elseIf;
    if (value < 0) {
      elseIf = "negative";
    } else if (value == 0) {
      elseIf = "zero";
    } else {
      elseIf = "positive";
    }

    String nestedIf;
    if (value > 0) {
      if (value % 2 == 0) {
        nestedIf = "positive even";
      } else {
        nestedIf = "positive odd";
      }
    } else {
      nestedIf = "not positive";
    }

    boolean shortCircuit;
    if (value > 0 && value < 10) {
      shortCircuit = true;
    } else {
      shortCircuit = false;
    }

    return Map.of(
      "simple_if", simpleIf,
      "else_if", elseIf,
      "nested_if", nestedIf,
      "short_circuit", shortCircuit
    );
  }
}
```

---

## Résumé

- `if` exécute un bloc si la condition est vraie
- `else` couvre le cas contraire
- `else if` permet plusieurs branches
- Les conditions peuvent être imbriquées
- `&&` et `||` utilisent le court-circuit logique
