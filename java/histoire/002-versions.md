# Versions

## Chronologie Java – dates précises et gouvernance

1. **23/05/1995** : Java 1.0  
   Créateur : Sun Microsystems  
   Lancement officiel du langage Java

2. **08/12/1998** : Java 2  
   Introduction des éditions :  
   J2SE (Standard), J2EE (Enterprise), J2ME (Micro)

3. **12/12/1999** : J2EE 1.2  
   Première version réellement utilisée en entreprise  
   EJB, Servlets, JSP

4. **24/09/2001** : J2EE 1.3  
   Généralisation des serveurs applicatifs

5. **11/11/2003** : J2EE 1.4  
   XML massif, SOAP, lourdeur accrue

6. **30/09/2004** : Java 5.0  
   Generics, annotations, enums  
   Base technique de Spring moderne

7. **11/05/2006** : Java EE 5  
   Abandon progressif du nom J2EE

8. **10/12/2009** : Java EE 6  
   CDI, annotations généralisées

9. **27/01/2010** : Oracle rachète Sun Microsystems  
   Changement de gouvernance Java

10. **18/03/2014** : Java 8  
    Lambdas, Streams, Optional, java.time  
    Tournant majeur du langage

11. **21/09/2017** : Java 9  
    JPMS (modules)

12. **25/09/2018** : Java 11 (LTS)  
    Nouveau cycle de releases Oracle (6 mois)

13. **14/09/2021** : Java 17 (LTS)

14. **19/09/2023** : Java 21 (LTS)

15. **2018** : Java EE transféré à Eclipse Foundation  
    Nouveau nom : Jakarta EE

16. Situation actuelle :  
    Java SE = socle universel  
    Jakarta EE = standards enterprise  
    Spring = implémentation dominante
